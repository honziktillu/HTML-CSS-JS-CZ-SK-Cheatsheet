display

Vlastnost display specifikuje jak bude element renderován

Může nabývat několika hodnot:

  • inline - zobrazí element jako inline element - vlastnosti jako width a height nebudou účinné - vezme si jen tolik, kolik bude potřeba pro obsah
  • block - zobrazí element jako block element - element začne na novém řádku a vezmi si celou šířku
  • table - zobrazí element jako tag <table>
  • none - element se nezobrazí
  • inherit - vlastnosti se zdědí od rodiče
  • ...

Editor

Pro živou ukázku je CSS a JavaScript vepsaný přímo v dokumentu - v praxi tyto soubory vždy oddělujte do externích souborů

<!DOCTYPE html> <html> <head> <style> p { width: 2000px; background: red; display: inline; } </style> </head> <body> <p> tento tag ma display na inline, takze nebere width v potaz </p> </body> </html>
