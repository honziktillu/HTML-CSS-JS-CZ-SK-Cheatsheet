outline

Vlastnost outline specifikuje zvýraznění nějakého elementu

Vlastnost outline je zkratkou pro vlastnosti outline-width - nastavuje šířku, outline-style - nastavuje styl, outline-color - nastavuje barvu

outline-style hodnoty:

  • none - defaultní hodnota - žádné zvýraznění
  • dotted - tečkované zvýraznění
  • dashed - čárkované zvýraznění
  • solid - plné zvýraznění
  • double - dvojté zvýraznění
  • ...

Editor

Pro živou ukázku je CSS a JavaScript vepsaný přímo v dokumentu - v praxi tyto soubory vždy oddělujte do externích souborů

<!DOCTYPE html> <html> <head> <style> button { outline: 5px dashed green; } </style> </head> <body> <button> ahoj svete </button> </body> </html>
