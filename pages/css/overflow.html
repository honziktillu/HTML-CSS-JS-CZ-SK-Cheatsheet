overflow

Vlastnost overflow se stará o obsah, který je větší než pro něj vyhrazené místo - ořízne část či přídá scroll bary

Může nabývat hodnot:

  • visible - defaultní hodnota - element není ořízlý, obsah se vyrenderuje mimo rozměry elementu
  • hidden - obsah je ořízlý - část obsahu je schovaná
  • scroll - obshaj je ořízlý - pro element se přidá automaticky scroll bar
  • auto - podobný hodnotě scroll, akorát přidává scroll bar jenom pokud to je nutné

K vlastnosti overflow patří ještě vlastnost overflow-x pro x-osu a vlastnost overflow-y pro y-osu
