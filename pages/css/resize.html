resize

Vlastnost resize specifikuje jestli a jak bude element roztahován od uživatele

resize hodnoty:

  • none - defaultní hodnota - uživatel nemůže roztahovat element
  • both - uživatel může roztahovat šířku a výšku
  • horizontal - uživatel může roztahovat šířku
  • vertical - uživatel může roztahovat výšku
