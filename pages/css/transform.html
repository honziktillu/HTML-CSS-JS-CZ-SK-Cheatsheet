transform

Vlastnost transform poskytuje 2D a 3D transformace pro elementy

transform hodnoty:

  • scale() - ovlivňuje rozměry elementu
  • skew() - naklání element
  • translate() - posouvá element
  • rotate() - otáčí element
  • matrix(n,n,n,n,n,n) - specifikuje 2D transformaci pomocí matrixu - kombinuje předchozí
  • matrix3d(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n) - specifikuje 3D transformaci pomocí matrixu - kombinuje předchozí
