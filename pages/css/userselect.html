user-select

Vlastnost user-select specifikuje výběr textu

user-select hodnoty:

  • auto - defaultní hodnota - text může být vybrán pokud to prohlížeč povoluje
  • none - zakáže výběr textu
  • text - text může být vybrán uživatelem
  • all - výběr text jde na jedno kliknutí místo dvou

Editor

Pro živou ukázku je CSS a JavaScript vepsaný přímo v dokumentu - v praxi tyto soubory vždy oddělujte do externích souborů

<!DOCTYPE html> <html> <head> <style> p { user-select: none; } </style> </head> <body> <p> text nelze vybrat </p> </body> </html>
