Slouží pro vytvoření hypertextového odkazu

<a> atributy:

  • href - specifikuje URL na kterou odkaz směruje
  • download - specifikuje, že bude přes odkaz stáhnut soubor - (stažení půjde jen pokud budou dodrženy webové bezpečnostní postupy)
  • target - specifikuje jak otevřít daný odkaz - _blank, _parent, _top, _self

Editor

<!DOCTYPE html> <html> <head> </head> <body> <a href="http://tillix.fun" target="_blank">Super mega odkaz</a> </body> </html>
