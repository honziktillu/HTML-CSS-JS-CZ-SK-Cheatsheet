Atributy

Všechny HTML elementy mohou mít atributy

Atributy poskytují informace o elementech a jsou vždy v otevírajícím tagu

Většinou jsou ve formátu jmenoatributu="hodnota"

Základní atributy

  • href - specifikuje URL na kterou daný element odkazuje - tagy a, base, link...
  • src - specifikuje URL pro externí zdroj - tagy img, audio, video, source, input, iframe...
  • alt - specifikuje alternativní text pro obrázek v případě, že by se obrázek nenačetl - tag img
  • style - slouží pro přidání CSS stylů pro element
  • lang - deklaruje jazyk webové stránky
  • title - dodatečné informace pro element - zobrazí se jako popisek po najetí kurzoru na nějaký element
  • id - specifikuje unikátní identifikátor pro HTML element - syntax id="hodnota"
  • class - specifikuje třídu pro HTML element, tato třída se využívá často pro selektory v CSS či odkazování v JavaScriptu - syntax class="trida1 trida2 trida3"
  • dataset - slouží pro ukládání vlastních hodnot - syntax: data-nasvybranynazev="hodnota" - v JavaScriptu se odkazuje na naší hodnotu ve stylu element.dataset.nasvybranynazev
