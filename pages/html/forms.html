TWS/WAP Cheatsheet

<form>

Slouží pro vytvoření formuláře

<form> atributy

  • action - specifikuje kam budou data z formuláře odeslána - URL
  • method - specifikuje, která HTTP metoda bude použita při odesílání dat - GET, POST
  • name - specifikuje jméno formuláře
  • autocomplete - specifikuje, jestli formulář bude poskytovat automatické vyplňování - on/off
  • enctype - specifikuje jak budou data z formuláře enkódována při posílání na server - platí jen pro POST

<input>

Slouží pro vytvoření jednořádkového pole ve kterém uživatel může zadat svoje údaje

<input> atributy

  • type - specifikuje typ vstupního pole - text, button, checkbox, date, email, file, hidden, image, number, password, submit...
  • autocomplete - specifikuje, jestli input bude poskytovat automatické vyplňování - on/off
  • autofocus - specifikuje, jestli input automaticky zvýrazněn po načtení stránky
  • form - specifikuje k jakému formuláři input náleží
  • maxlength - specifikuje maximální počet znaků
  • minlength - specifikuje minimální počet znaků
  • value - specifikuje hodnotu, kterou bude input obsahovat
  • name - specifikuje jméno inputu
  • required - specifikuje, že input musí být vyplněn před odesláním
  • placeholder - specifikuje text nápovědy v pozadí inputu

<select>

Slouží pro vytvoření rozevírajícího seznamu (drop-down list)

<option>

Slouží pro vytvoření položky v drop-down seznamu

<label>

Slouží pro vytvoření jmenovky pro elementy typu input

<textarea>

Slouží pro vytvoření vícerořádkového vstupního pole ve kterém uživatel může zadat svoje údaje

<textarea> atributy

  • name - specifikuje jméno elementu textarea
  • placeholder - specifikuje text nápovědy v pozadí elementu textarea
  • required - specifikuje, že element textarea musí být vyplněn před odesláním
  • maxlength - specifikuje maximální počet znaků
