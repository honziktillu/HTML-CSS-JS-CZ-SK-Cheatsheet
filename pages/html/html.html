TWS/WAP Cheatsheet

<!DOCTYPE html>

Deklarace typu dokumentu - říká prohlížeči, že má očekávat dokument typu HTML

Tag html

HTML tag, který specifikuje, že dokument je typu HTML - kořen HTML dokumentu - hlavní obal pro ostatní elementy

Tag head

Obal pro metadata - dodatečné informace k HTML dokumentu

Tag body

Obal pro různé elementy jako jsou různé odstavce, nadpisy, obrázky, odkazy...

Tag p

Značení pro vytvoření odstavce

Editor

<!DOCTYPE html> <html> <head> </head> <body> <p>Ahoj světe</p> </body> </html>
