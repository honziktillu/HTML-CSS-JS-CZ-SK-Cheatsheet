<img>

Slouží pro vložení obrázku do HTML stránky

<img> atributy

  • src - cesta k obrázku (source)
  • alt - alternativní text k obrázku pro případ, že by se nenačetl
  • width - šířka obrázku v px
  • height - výška obrázku v px

Editor

<!DOCTYPE html> <html> <head> </head> <body> <img src="../../res/img/icon.png" alt="icon.png" width="200" height="200"> </body> </html>
