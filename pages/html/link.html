TWS/WAP Cheatsheet

<link>

Definuje propojení mezi současným dokumentem a externím zdrojem

Často využívaný pro připojení CSS souborů a ikon

<link> atributy

  • crossorigin - specifikuje jak element řeší crossorigin požadavky. Nabývá hodnot anonymous (CORS požadavky budou mít vlajku nastavenou na 'same-origin') a use-credentials (CORS požadavky budou mít nastavenou vlajku na 'include')
  • href - specifikuje umístění připojeného souboru - URL
  • rel - specifikuje vztah mezi současným dokumentem a připojeným souborem - icon, stylesheet, author, preload...
  • type - specifikuje media typ připojeného souboru

Editor

<!DOCTYPE html> <html> <head> <link href="../../res/css/style.css" rel="stylesheet" type="text/css"> </head> <body> <p class="text-sample">Ahoj světe</p> </body> </html>
