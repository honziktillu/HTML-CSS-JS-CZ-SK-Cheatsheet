<svg>

Slouží k vytvoření obalu pro SVG grafiku

SVG obsahuje elementy pro kresbu cest, kruhů, textů, obrázku...

<svg> atributy:

  • width - šířka svg elementu
  • height - výška svg elementu

<svg> elementy:

  • <rect> - vykreslení obdelníku
  • <circle> - vykreslení kruhu
  • <line> - vykreslení čáry
  • ...
