JavaScript

JavaScript je scriptovací jazyk pro webové technologie. Můžeme v něm dělat jak scripty na straně klienta, tak psát v něm logiku serveru (NodeJS)

Patří k nejpopulárnějším jazykům světa (Stackoverflow survey 2020 - JavaScript 67% - 1. místo, PHP 26% - 8. místo)

Hello World

JavaScript se dá psát do tagu <script></script>

Tento tag se většinou píše buď do tagu head nebo před koncový tag body

Pro lepší správu se scripty oddělují do separátních souborů většinou s koncovkou .js

Následně se na tyto scripty odkazuje pomocí <script src="/cesta/k/souboru/soubor.js"></script>

První věcí co se učí v JavaScriptu je funkce console.log(); - výpis do konzole (tu najdete v DevTools - např. Chrome - F12 - Console)
