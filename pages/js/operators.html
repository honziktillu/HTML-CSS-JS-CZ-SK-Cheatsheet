TWS/WAP Cheatsheet

Operátory

Aritmetické:

  • + - sčítání
  • - - odčítání
  • * - násobení
  • / - dělení
  • % - modulo (zbytek po dělení)
  • ++ - inkrement - přičte 1
  • -- - dekrement - odečte 1

Přiřazení:

  • = - nastaví proměnnou na nějakou hodnotu - let variable = 5;
  • += - syntax: a += b; - zkracený zápis pro a = a + b
  • -= - syntax: a -= b; - zkracený zápis pro a = a - b
  • *= - syntax: a *= b; - zkracený zápis pro a = a * b
  • /= - syntax: a /= b; - zkracený zápis pro a = a / b
  • %= - syntax: a %= b; - zkracený zápis pro a = a % b

Relační:

  • == - syntax: a == b - jestli po konverzi se hodnota a rovná hodnotě b
  • === - syntax: a === b - jestli se hodnota a rovná hodnotě b a mají stejný datový typ
  • != - syntax: a != b - jestli po konverzi se hodnota a nerovná hodnotě b
  • !== - syntax: a !== b - jestli se nerovnají hodnoty nebo datový typy
  • > - syntax: a > b - jestli je a větší jak b
  • < - syntax: a < b - jestli je a menší jak b
  • >= - syntax: a >= b - jestli je a větší nebo rovno b
  • <= - syntax: a <= b - jestli je a menší nebo rovno b

Logické:

  • && - logický AND - jestli platí levá a pravá strana
  • || - logický OR - jestli platí levá nebo pravá strana
  • ! - logický NOT - negace
